1. Basic Data Binding

Create a two-way binding on a form input element or a component.

Read More

Here we can make use of the v-model directive by attaching it to the input element on line 3. By doing this it makes a connection with our data object property of the same name. In this case message: ‘This a message’ is attached to v-model=”message”.

The value within the H1 is also displaying the value of message by the use of Declarative Rendering to the dom by use of a simple template syntax {{ }}. Ex. {{ message }}

Try it out :

Change the data property named message to anything you like and watch the value within the input and H1 change dynamically. This is also a two way binding* , meaning that whatever is typed within the input will overwrite the data property named message. The checkbox on line 8 as well is attached to the data property named checked and returns the value to the label on line 9 by the way of declarative rendering using the template syntax.

*This can be viewed by using the inspector tools.
